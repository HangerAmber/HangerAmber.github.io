# Hange Zhou Personal Website

This repository contains a static personal website prepared for GitHub Pages.

## Files

- `index.html`: page structure
- `styles.css`: visual design and responsive layout
- `script.js`: data-driven content, bilingual toggle, and particle animation
- `assets/`: profile photo and award images

## Deployment

1. Create or use a GitHub repository.
2. Upload all files in this directory to the repository root.
3. Go to `Settings` -> `Pages`.
4. Set the source to:
   - Branch: `main`
   - Folder: `/root`
5. Save and wait for deployment.

If the repository is named `HangerAmber.github.io`, the site will usually be available at:

`https://hangeramber.github.io/`

## Content Updates

- Replace the profile photo in `index.html` if needed.
- Update publications, patents, projects, awards, and timeline entries in `script.js`.
- Award image paths are managed in `script.js` using the `images` field.
