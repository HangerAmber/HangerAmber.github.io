前言

此文是笔者学习MySQL数据库时用到的一些基础命令行，在此记录一下，方便回忆，如果能给您带来些许帮助也是极好的。

用管理员身份打开cmd net start mysql 启动数据库

基本命令

  1. 进入到mysql安装路径的bin目录下

  2. 打开数据库命令：mysql -uroot -p

  3. 停止数据库命令：net stop mysql
  4. 查看版本 select version();
  5. 查看当前时间 select now();
  6. 远程连接 mysql -h ip地址 -u 用户名 -p

数据库操作

  1. 创建数据库
    • 格式：create database 数据库名称 charset=utf8;
  2. 删除数据库
  • 格式：drop database 数据库名：
  1. 切换数据库
    • 格式：use 数据库名
  2. 查看当前选择的数据库
  • 格式：select database();
